# Portfólio Pessoal

Este repositório contém o código-fonte do meu portfólio pessoal, desenvolvido com o objetivo de apresentar meus projetos, habilidades e evolução na área de desenvolvimento de software.

---

## Sobre o projeto

Este projeto consiste em uma aplicação web estática desenvolvida com foco em organização, responsividade e apresentação visual de projetos.

O portfólio reúne trabalhos práticos desenvolvidos durante meus estudos, incluindo aplicações web e projetos acadêmicos.

---

## Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript
